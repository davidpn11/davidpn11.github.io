Projetos que eu já participei

Primeira Iniciação Científica

Neste projeto, tive que dar auxílio tecnológico para o grupo Ciência,Café e Cultura, um projeto do CEFET-MG que criou um Café Científico, um encontro que para fazer debates entre especialistas de sobre um determinado tema. Era bem legal! Meu trabalho foi desenvolver ferramentas para auxiliar esses encontros e por desenvolver uma rede social para que o público tivesse uma maior conectividade com o café científico, que foi a primeira vez que tive contato com linguagens web, como HTML,CSS,Javascript, JQuery, PHP,etc. Infelizmente essa rede social falhou miseravelmente, mas as ferramentas continuam sendo utilizadas e elogiadas, então nem tudo foi perdido =).

Bom Amigo - Rede para adoção de animais domésticos

Esse foi um trabalho para a Disciplina de Engeharia de Software I. Nele desenvolvemos uma rede onde pessoas publicam seus animais que querem disponibilizar para adoção, e outra pessoa pode vizualizar informações sobre ele e entrar em contato com o dono. Tudo foi desenvolvido em Android, e ficou bem interessante o aplicativo. Infelizmente não ficou tão bom(nem tão otimizado) para disponibilizarmos na Google Play mas foi uma boa maneira de aprender e treinar o desenvolvimento Android.

Segunda Iniciação Científica

Minha segunda iniciação científica(que estou atualmente) é enrelação ao paralelismo, onde devemos paralelizar uma programa serial que tem uma complexidade fatorial. Tudo está sendo desenvolvido em C CUDA( Linguagem da NVIDIA) e está sendo uma esperiência muito legal para aprender mais uma nova linguagem.

Perceptron

Para o curso de Inteligência Artificial estamos desenvolvendo um Percetron, isto é, uma rede neural artificial com aprendizado para a partir de uma base de conhecimento, conseguir detectar a partir de dados de pacientes a probabilidade de uma pessoa ter cancêr. Tudo está sendo desenvolvido no matlab.

Coloração de Grafos utilizando a Heurística GRASP

Outro projeto que desenvolvido em matlab, foi criado um programa para verificar o número mínimo de cores necessário para o problema de coloração de grafos, que é quando cada nó de um grafo deve ter uma cor e nós adjacentes não podem ter a mesma cor. Por ser um problema com complexidade alta, foi utilizado a Heurística GRASP para encontra uma solução boa para o problema.
